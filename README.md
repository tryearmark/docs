# Earmark Product Docs (Mintlify)

This repository contains product-focused Earmark documentation hosted on Mintlify.

## Run locally

```bash
cd /Users/sanden/Developer/docs
npm install -g mint
mint dev
```

Open `http://localhost:3000`.

## Validate docs

```bash
mint broken-links
```

## Content focus

The docs are intentionally product-centric and written for delivery teams:

- Onboarding and first-run workflows
- Core product features
- Practical meeting playbooks

API and implementation-level technical references are intentionally excluded from this repository scope.
